Frameworks CSS:

“Son herramientas usadas que permiten encapsular las tareas repetitivas en módulos genéricos fácilmente reutilizables, permiten al diseñador centrarse en los elementos únicos de cada diseño en los que puede aportar valor”

¿Para qué se utilizan?

Permiten entregar proyectos en menor tiempo y con un código más limpio, cuya eficacia ya ha sido comprobada a partir de los programadores pueden complementar y/o modificar la estructura base para que el software o aplicación para que cumpla con los objetivos requeridos.

Ventajas de los frameworks

•Disminucion de tiempo ya que se dispone de un esqueleto sobre el cual se desarrolla la aplicacion
•Facilita desarollos colaboativos yaque deja definidos los estandares de programación
•Es facil encintrar herramientas, módulos e información para utilizarlo
•Proporciona mayor segurtidad ya que tiene una buena parte de las vulnerabilidades resueltas
•Por lo general hay comunidades que ayudan a resolver consultas.

¿Qué tipos existen?

Algunos de ellos son:

Para aplicaciones Web

Son usados para la creacion de proyectos OnLine, desde el diseño web de una pagina hasta servicios web mucho más complejos

Para aplicaciones en general

Permite complementar la estructura de una aplicacion para un sistema operativo

Para tecnología AJAX

Permite que el usuario haga solicitudes al servidor sin que sea necesario recargar una página despues de cada nueva solicitud, exixten frameworks especificos para estetipodetecnológia, permitiendo la reutilizacion de un código ya elaborado

De gestion de contenidos

Facilita la programacion de aplicaciones de un Sistema de Gestion de Contenidos, como WordPress.

De multimedia

Facilita el trabajo de los programadores que se basan en videos, imagenes y audios y colabora con la creacion de aplicativos multimedia en general

Tres de los mejores frameworks son:

1. Bootstrap:

Se le atribuye el mérito de incluir el diseño receptivo a gran escala, fue el primero en promover la filosofía de “dispositivos móviles primero”

Lo bueno:

-Permite la creación rápida de prototipos: Sin necesidad de preocuparse por posicionamiento CSS o compatibilidad con el navegador, con escribir en HTML y aplicar las clases de CSS apropiadas hace que la capacidad de respuesta cobre vida
-Gran ecosistema: Tiene el ecosistema mas grande de marcos de front-end La cantidad de diseños de sitios web, temas, paneles de administración, componentes de la interfaz de usuario, etc., creados con Bootstrap es alucinante y sigue mejorando
-Una gran colección de componentes: Bootstrap ofrece, listos para usar, casi todos los componentes de la interfaz de usuario que probablemente necesite. Navegación, formularios, tarjetas, modales, botones, insignias, barras de progreso, alertas, etc

2. Foundation:

El marco de front-end más receptivo del mundo, es intuitivo y fácil de memorizar

Lo bueno:

-Herramientas completas: Tiene como objetivo solucionar casi todo tipo de problemas de front-end
-Flexibilidad extrema: El diseñador tiene control total de sus interfases, no impone ningún estilo.
- Más que solo componentes de la interfaz de usuario: Los desarrolladores han incluido un sistema avanzado de imágenes receptivas, un componente de tabla de precios, validación de formularios, soporte de derecha a izquierda, incrustaciones receptivas y más

3. Bulma:

Es relativamente nuevo en el mundo de los marcos CSS, pero se ha posicionado rápidamente; este no tiene componentes de JavaScript, se ha basado estrictamente en la sintaxis CSS y valores predeterminados elegantes

Lo bueno

-Muy popular
-Clases extremadamente legibles: Tiene un sistema simple y poderoso para crear cuadriculas, llamadas mosaico
-Elegante

Otros ejemplos de frameworks son:

•.Net: es Framework de Microsoft y uno de los más utilizados. viene instalado con el sistema operativo Windows

•Symphony: proyecto PHP de software libre.

•Zend Framework: framework de código abierto para desarrollar aplicaciones web y con servicios web PHP.

•Laravel: uno de los frameworks de código abierto más fáciles de asimilar para PHP.

•Django: framework de desarrollo web de código abierto escrito en Python.

•Ruby on Rails: framework de aplicaciones web de código abierto escrito en el lenguaje de programación Ruby.

•Angular: framework de código abierto desarrollado en TypeScript y mantenido por Google.
