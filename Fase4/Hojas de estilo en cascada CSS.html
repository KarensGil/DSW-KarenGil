Hojas de estilo en cascada (css)

Definición: Es el formato recomendado para las paginas escritas en formato HTML, la idea que se encuentra detras del desarrollo de CSS es separar la estructura de un doumento de su presentación o aspecto, Ofrecen un modo flexible para agregar estilos a un sitio web Este conjunto de reglas de formato rige el aspecto de las páginas y permite definir las fuentes, los colores, el formato y otras características de presentación.
Imagen de autoria propia

¿Para qué sirven las hojas de estilo?

"Constituyen el complemento ideal para HTML o XHTML, su misión es definir la apariencia y el estilo de sus elementos" Diseño de páginas Web con XHTML, JavaScript y CSS. Grupo Editorial RA-MA.

Algunas de las cosas que se pueden hacer usando este lenguaje, dentro de las cuales podemos encontrar:
*Permite definir toda la pagina Web de una sola vez
*Aplicar estilos visibles en un trozo de la pagina
*definir incluso varios estilos diferentes para una sola etiqueta
*permite aplicar formato de una forma más exacta
*Una vez aparece la sintaxis CSS se vuelve más fácil maquetar las paginas, ya que las herramientas dadas son mucho más prácticas.




Declaraciones, propiedades y valores

Una declaración consiste básicamente en la asignación de un valor a una propiedad, en otras palabras y para comprender mejor el concepto, la declaración responde preguntas como ¿Qué tamaño de fuente debería tener el título?,¿Qué fondo debería tener?, etc. El formato de la declaración está dado por una propiedad seguida de dos puntos y un valor, finaliza con punto y coma (ej. font-family: 'Times New Roman', Times, serif; para definir la fuente de la página)

Selectores

Son los elementos que comienzan la regla, hace referencia a un grupo de uno o más elementos HTML donde se aplican las declaraciones CSS

Tipos de selectores

Selector de elemento: Selecciona todos los elementos HTML del tipo especificado
Selector de identificación: Selecciona el elemento en la página con ID especificado
Selector de clase: Selecciona los elementos de la página con una clase especificada
Selector de atributo: Elementos en una página con el atributo especificado
Selector de pseudoclase: selecciona elementos especificados cuando por ejemplo el puntero esté sobre el

Datos adicionales:

*Cada regla debe estar encapsulada en llaves ({}).
*Dentro de cada declaración deben usarse dos puntos (:) para separar la propiedad de su valor
*Dentro de cada regla se usa el punto y coma (;)
